Test Your Luck!



This is a website that assumes you are a passenger on the Titanic Ship that tragically sunk. Under the hood it uses a random forest classifier which has been trained on data already available about the passengers to predict if you will survive or not.

There are mainly three places from where the passengers embarked from:

  • Southampton
  • Cherbourg
  • Queenstown

The title data helps the model better understand the characteristics of the passenger.
